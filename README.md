# LT Inventory Tracker

LT Inventory Tracker is a modern web app for managing inventory at LT Telehealth.  
It allows users to securely log in, track and update inventory items (laptops, peripherals, shipping, and cleaning supplies), save and view past summaries, and export data to Excel.  
The app features a clean, responsive interface and ensures that each user's data is private.

**Access the app here:** [Live Demo](https://your-vercel-url.vercel.app)

---

**Features:**
- Secure login and signup
- Add and edit inventory counts and minimums
- Save summaries and view past entries
- Export inventory data to Excel
- Delete old summaries if needed

**How it works:**  
Log in or sign up, enter your inventory data, save summaries, and manage your inventory efficiently.  
Past summaries are only visible to the logged-in user.

---

*Built with React, Supabase, and Tailwind CSS.*
